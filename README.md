# todolistsite
This is a site designed for myself to optimize my scheduling and to do list.

## What this tool does

This to-do list app helps you keep track of tasks in one place. You can add tasks, set their priority, mark them as complete, edit or delete them, filter by status, and click a task to store extra details like date, time, location, and notes.

## How to use it

Open https://lvandervlugt123.github.io/todolistsite/ in your browser.

- Type a task into the input field and click **Add**.
- Choose a priority level: High, Medium, or Low.
- Click the checkbox to mark a task complete.
- Use the filter buttons to view all, active, or completed tasks.
- Click a task name to open its detail popup and add information like time or location.
- Use the edit and delete buttons to update or remove tasks.
- Click **Reset** to clear everything and start over without refreshing the page.
